Cada elemento HTML puede llevar al atributo id. Se utiliza para identificar de forma unica ese elemento de otros elementos de la pagina. Su valor debe comenzar con una letra o un guion bajo (no un número o culaquier otro caracter). Es importante que no hay dos elementos en la misma pagina tienen el mismo valor para sus atributos ID (de lo contrario, el valor ya no es el unico)

dar a un elemento una identidad unica le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la pagina

Por ejemplo, es posible que desee asignar un parrafo dentro de la pagina (quizas un parrafo que contiene una cotización de extracción) un estilo diferente que todos los demas parrafos. En el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayusculas usando CSS.
